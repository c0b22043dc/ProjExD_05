#  断罪のこうかとんクラッシャー　-轟翼斬鳥の襲撃
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1
## ゲームの概要
車を走らせ、近づいてくる障害物を避けるゲーム
## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
### 担当追加機能
背景のスクロール、障害物の当たり判定、出現
